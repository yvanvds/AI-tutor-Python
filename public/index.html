Install the AI Tutor for Python

A classroom-focused app that helps students practice Python with guided exercises, instant feedback, and teacher-controlled progress tracking.

Download installer (Windows) Read installation notes

The download is a single .exe and should start immediately. If your browser blocks it, choose “Keep” or “Keep anyway”.

Who can use it?

Students can install the app freely. Sign-in will only work after your teacher enables your account (they toggle access for your API key in the teacher dashboard). You don’t need prior permission to install—just know that the app becomes usable once your teacher has switched you on.

Windows SmartScreen

Because this is a new, unsigned app, Windows may show a warning that the publisher is unknown. If you trust this source, click More info → Run anyway to continue.

What it does

Generates tailored Python exercises, checks code, and provides hints—aligned with your class goals and controlled by the teacher’s settings.
